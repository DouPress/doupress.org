DouPress

开源、简洁、高效的博客程序，采用文本数据库存储，无需数据库支持，上传即用，备份方便。

快速开始 查看源码

核心特性

📦

无需数据库

只要求 PHP 环境，文本数据库存储，上传即可运行，摆脱数据库依赖。

👤

个人博客设计

专为个人网站打造，剔除复杂的成员管理和权限设置，专注内容创作。

🏷️

标签代替分类

只有标签没有分类，免除发布文章时该如何分类的纠结。

✏️

极简内容系统

只有"文章"和"页面"两个系统，没有评论、插件等干扰，让你更专注于创造内容。

🔄

备份还原简便

下载上传全部文件即可完成备份和还原，所有文章配置完整保留。

🎨

多主题支持

支持多主题切换，可自定义网站外观，满足个性化需求。

项目结构

/src/
├── index.php        # 入口文件
├── admin/           # 后台管理
├── core/            # 核心
├── data/            # 内容数据
│   ├── config.php   # 配置文件
│   ├── posts/       # 文章
│   └── pages/       # 页面
├── theme/           # 主题目录
└── install.php      # 安装文件

快速安装

  1. 1

    下载安装脚本

    下载最新版的 install.php，上传到网站根目录

  2. 2

    访问安装页面

    浏览器访问 http://example.com/install.php

  3. 3

    完成安装

    填入网站信息和初始账号密码，开始使用

构建安装包：

php build.php [版本号]
